Problem statement

As warehouses are getting bigger with scaling heights, inventory management/auditing is becoming a huge problem. Today in warehouse package fetching and Placing is almost automated but still there is no automated solution for inventory auditing. Well a video can explain it better!!



Solution

Warehouse is not a drone friendly environment, it brings many challenges which are considered not significant in open space. For example, Drone navigation system relies on GPS, and GPS signals are very weak, they cannot penetrate warehouse walls/roof. To put in simple terms, itâ€™s a GPS denied environment. On top of that, GPS possess 1-5-meter variance which is quite significant in indoor environment. So basically, to identify a location we need to switch to local reference navigation which in turn requires area mapping i.e 3D SLAM. Secondly, drone is highly prone to collision specially in indoor environment, it needs environment sensing from all 6 directions (left, right, front, back, top, bottom).

Technology Stack:

  • ML: For object classification
  • Stero Camera: To make point cloud and SLAM
  • ROS: Path planning and navigation i.e move_base
  • Azure: cloud storage
  • OpenCV: Barcode scanning
  • DJI ROS SDK
  • Android app: For user control, live feed, and inventory display
  • Nvidia TX2: onboard heavy computing
