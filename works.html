Huron, a webapp to monitor search nodes

This is a web application which provides scalable solution to monitor the search nodes. Multiple Java Server Pages were built to form an application package which is used to monitor the live nodes of Elasticsearch, an enterprise search engine. The nodes will be queried at regular default intervals to know the state of each and every node in the cluster.
Technologies used : Socket programming, Java Server pages, HTML, CSS
